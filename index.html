⚙ SYSTEM CONFIGURATION
▸ GOOGLE CALENDAR

Google Calendar requires OAuth via the Google API. Follow these steps to get your API key:

1.Go to console.cloud.google.com → Create a new project
2.Enable the Google Calendar API for your project
3.Go to Credentials → Create → API Key. Restrict it to Calendar API only.
4.Also create an OAuth 2.0 Client ID (Web App). Add http://localhost and the URL of this page as authorised origins.
5.Paste your Client ID and API Key below, then click Connect.
▸ CANVAS LMS

Canvas has a student-accessible API. Get your token from Canvas account settings:

1.Log into Canvas → click your profile pic → Account → Settings
2.Scroll to Approved Integrations → click + New Access Token
3.Give it a name like "Nexus Dashboard", no expiry, copy the token
4.Find your Canvas domain — it's the base URL e.g. canvas.myuni.edu.au
5.⚠ CORS Note: Your uni's Canvas instance may block direct browser API calls. If it fails, see the proxy workaround below.
▸ CORS WORKAROUND (if direct fails)
If your uni blocks the API call, install the CORS Unblock browser extension, or run a simple local proxy with: npx cors-anywhere on port 8080 and prefix your domain with http://localhost:8080/
